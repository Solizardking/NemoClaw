# Nemo Clawd Catalog Skills

This directory is the optional NVSkills CI watched location for customer-facing skill publication.
Nemo Clawd keeps source skills under `.agents/skills/` and copies publishable customer-facing skills here.
Do not put generated documentation copies here.

Nemo Clawd currently maintains one customer-facing source skill:

- `skills/nemoclawd-user-guide/` for Solana-native `nemoclawd` installation, MCP, and sandbox guidance.

When publishing or refreshing it in the catalog, follow `.github/catalog-skills-signing-flow.md`.
